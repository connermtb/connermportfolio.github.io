Unit 1

Moving Guy

Description

We were tasked with making the guy move in this project using the WASD keys. We had to make it so that when one of those keys were pressed, the guy would move in a certain direction. This lab was challenging as I kept having errors because I was still learning how to make the class file interact with the runner. Eventually though, I learned how to make it interact and how to make the guy move if a certain key was pressed.

Concepts Learned

  • How to use if statements
  • How to make a class and runner work together
  • How to program keys to make an object move

Unit 2

Line Runner

Description

We were tasked with completing a side crolling program where you had to jump over incoming objects. After we got the program to work, we had to add our own objects into the game. We had to add one that increased the score and one that decreased it. I added in a bomb and a coin. The coin is Obama's face because he is awesome. To create the bomb and coin we had to create two new classes from scratch. We also had to make the objects affect the score mechanism to either detract from it or increase it.

Concepts Learned

  • How to use if statements
  • How to create a class from scratch
  • How to make objects detract or add from score

Unit 3

Coin/Background Game

Description

At first, we were tasked with creating a background from only 2 images. We had to make the images repeat until the whole background was covered by the images. After that, we had to put our person in from unit 1 and make him work on the screen. We also made boulders appear for aesthetic. Then we had to program in a coin and a timer. This made it so that you had to collect a certain amount of coins before the timer ran out. The coin would appear in random places and the player would have to hurry to collect 10 coins in 10 seconds.

Concepts Learned

  • Random placement of objects
  • While true loops
  • Repeating bits of code

Unit 4

Costume Guy

Description

We were tasked with finding a sprite sheet and then making that sprite sheet work so that the sprite looked animated. We had to make a list of costumes and corilate the items in that list to appear when a certain key was pressed. For example, if the left key was pressed, the costume that made the guy look like he was walking to the left would be called and would appear on the screen. This lab was ery interesting to me and I would later go on to use the same code in my final project.

Concepts Learned

  • How to write and use lists
  • How to create and use a sprite sheet
