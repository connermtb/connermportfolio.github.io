Name Assignment

Description

In this project, I used individual sprites to spell out my name and a sprite that represented something I liked. I chose a taco because I like tacos. When the flag was clicked, the sprites had to display 3 different behaviors. For extra credit, I had to add some code that would make all of the letters reset to their original places.

Concepts Learned

  • Using event blocks like “when green flag is clicked”
  • Using forever loops
  • Using motion blocks so that the different sprites would move
  • Using the looks block to have the letters change colors rapidly
  • Sprites used to spell out my name and for the taco

Computer Buying Project

Description

We were tasked with finding a computer for James’ brother. He was going to be using the computer to animate and he had a budget $1600. The computers that we chose for him were made by razer, dell, lenovo, and mac. We ended up choosing the dell as it offered the best performance for the price.

Concepts Learned

  • Learned what all of the computer components did and how to compare them
  • Learned how to use if statements
  • Learned how to use speech bubble to make the sprite communicate
  • Learned how to use the “when touching color” block

Final Project

Description

For the final project, we were tasked to either create a game or story in scratch. I chose to create a game. The game I created was a side scrolling motorcycle game. You had to collect the green dots while avoiding the red dots on the screen. It worked pretty well and kept score.

Concepts Learned

  • Learned how to import a background from the internet
  • Learned how to create a scrolling background
  • Learned how to use variables to keep score
  • Learned how to make a sprite jump while not affecting the x-axis movement
  • Learned how to import a sprite from a file into the program
